# PaletteForge Changelog

## v0.1.4 - Auto Match Engine

### Added
- Added `core/swapper.py`.
- Added `PaletteMatcher` engine.
- Added brightness-based source-to-target color matching.
- Added RGB nearest-color fallback.
- Added Auto Match button to the sidebar.
- Added Auto Match Mapping viewer to the Palette Explorer.
- Added sidebar status updates.

### Changed
- Updated app title and sidebar version from v0.1.3 to v0.1.4.
- Reduced palette scroll heights to make room for the mapping viewer.
- Kept GIF playback, dual preview, and palette extraction intact.
